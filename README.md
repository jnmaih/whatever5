# Demo NGINX CNCF NBG CI/CD Template

This template generates a minimal demo-nginx-cncf-nbg service with TechDocs directory and a GitHub Actions CI workflow.

Generated repository structure:

- catalog-info.yaml
- README.md
- mkdocs.yml
- docs/index.md
- docs/setup.md
- .github/workflows/ci.yaml
- app/ (container sample, placeholder)

Once scaffolding completes, the component is automatically registered in Backstage and appears in the catalog.
